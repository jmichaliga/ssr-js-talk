  • 2010: npm - node package manager
  • 2012: #NewTwitter & IsoMorphic JS

SSR Traditionally Consists of:

  • Server: or rendering FrontEnd, but also not nessacarily Node. There are other projects, Nashorn (Oracle), php-v8js (you can run JS from PHP) too.
  • Client: Library for Rendering (for our case, this is Vue.js)
  • External APIs still exist, and are typically separated from the rendering server.
